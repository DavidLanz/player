player
=====
一款强大的视频播放器，基于ckplayer6.3用java实现视频解析无广告播放，目前支持优酷、搜狐、乐视等。，详情见：http://www.terwer.com/special.html#用Java实现视频地址解析

相关技术
======
>1、maven项目管理     
>2、spring mvc    
>3、jstl    
>4、Apache httpclient
