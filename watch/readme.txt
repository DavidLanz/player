[服务器配置要求]

1. 兼容Windows或Linux + Apache搭建的PHP环境；Windows + IIS的PHP环境未测试；其他环境未测试。
2. PHP环境至少支持cURL或socket之一。
3. 因使用了json方法，PHP版本要求>=5.2.0。



[API参数说明]

-- 视频资源API --

1. 入口文件为play.php，该文件可自由更名。
2. [vid]参数：视频资源ID，各个视频提供商的视频ID并未有统一格式，如何取得视频ID不赘述。
3. [vtype]参数：视频类型即来源
   56视频：56
   网易视频：163
   凤凰视频：ifeng
   酷六视频：ku6
   乐视网：letv
   PPS在线电视：pps
   新浪视频：sina
   搜狐视频：sohu
   土豆网：tudou
   优酷网：youku
   腾讯视频：qq
4. [ver]参数：此参数决定了XML的文件输出格式，本程序兼容ckplayer 5.9及以下使用旧格式的XML和ckplayer 6.0及以上的新格式XML。
   兼容ckplayer 5.9及以下版本，请提供该参数的值为5即ver=5，兼容ckplayer 6.0及以上版本，请提供该参数的值为6即ver=6。
   
   *[ver]参数为可选，程序默认输出兼容ckplayer 6.0及以上版本的XML。

-- 电视直播资源API --

1. 入口文件为live.php，可自由更名。
2. [tv]参数：可选电视频道列表如下：  
   中文形式：四川卫视，浙江卫视，湖北卫视，东方卫视，香港卫视，安徽卫视，深圳卫视，珠江卫视，广东卫视，辽宁卫视，青海卫视
	     重庆卫视，山东卫视，陕西卫视，云南卫视，贵州卫视，河南卫视，天津卫视，广西卫视，东南卫视，甘肃卫视，宁夏卫视
	     内蒙古卫视，健康卫视，英语辅导，广东公共，广东新闻，嘉佳卡通
   简写形式：sctv，zjtv，hbtv，dftv，hktv，ahtv，sztv，zhjtv，gdtv，lntv，qhtv
	     cqtv，sdtv，sxtv，yntv，gztv，hntv，tjtv，gxtv，dntv，gstv，nxtv
	     nmgtv，jktv，yyfd，gdgg，gdxw，jjkt

   *部分浏览器（如火狐）不支持中文参数传递（自动url编码），请使用非中文形式
   *程序亦支持其他形式（数字ID，全拼）的参数，请参阅程序源码，此处不列举
   *直播信号取自腾讯直播，除此列举电视频道外，暂不支持其他频道

[使用示例]

-- 视频资源 --

1. 输出兼容ckplayer 5.9及以下版本的XML列表：
   http://www.yoursite.com/play.php?ver=5&vtype=youku&vid=XNTY2NDEzNDU2
   
2. 输出兼容ckplayer 6.0及以上版本的XML列表
   http://www.yoursite.com/play.php?ver=6&vtype=tudou&vid=170070280
   或
   http://www.yoursite.com/play.php?vtype=tudou&vid=170070280
   

-- 电视直播资源 --

http://www.yoursite.com/live.php?tv=广东卫视
或
http://www.yoursite.com/live.php?tv=gdtv

*以上示例http://www.yoursite.com/为你的站点程序路径，请视自己的实际情况修改
